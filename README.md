# How to run Streamlit by yours trully
## Setup environment
```
conda create --name main-ds python=3.11
conda activate main-ds
pip install numpy pandas  matplotlib seaborn jupyter streamlit 
```

## Run steamlit app
```
streamlit run dashboard.py
```
